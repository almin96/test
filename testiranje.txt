U ovaj text file pisat cemo sve korake koje sam naucio u gitu:

1. Initialization of the folder: git init
2. Commit: git commit / git commit -m "added filename.md"
3. Dodavanje file-a u pripremnu branchu/prije commit-a: git add filename.md
4. Provjeravanje statusa: git status
5. Dodaj sve fajlove kao commit: git add . 
6. Kreiraj novu branchu: git checkout -b new
